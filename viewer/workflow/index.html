Step 2 / 5

第一轮评测（在 Agent 中执行）

请在 Agent 对话中完成「触发 + 断言」类评测（元 skill / skill-creator 标准流程）。

  • 为每条用例并行跑 with_skill 与 without_skill，并用 grader 检查 assertions。
  • 工作区下出现 eval-* 及阶段一产出（如 response.md、grading.json）。第一轮通过后再进入第二轮。
